# Changelog

All notable changes to Generic Design System will be documented in this file.

## [0.1.0] - 2026-07-04

### Added
- Initial release of Generic Design System (GDS)
- 70+ Lit Web Components across 11 categories
- 3 neutral themes: Neutral Light, Neutral Dark, Neutral Warm
- CLI with agent docs generation (`gds init --features agents`)
- Component documentation command (`gds component --list`, `gds component Button`)
- Template scaffolding (`gds template --list`, `gds template dashboard --skeleton`)
- Documentation commands (`gds docs theme|tokens|styling|layout`)
- 6 page templates: dashboard, settings, auth, profile, chat-ui, data-table
- Vite docs site with interactive component showcase
- Demo app with dashboard preview
- Agent-ready docs for Claude, Cursor, and Codex
- Design token system with CSS custom properties
